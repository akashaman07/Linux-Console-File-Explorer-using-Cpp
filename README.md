# Linux Console File Explorer using C++

## Description
This project is a Linux Terminal based File Explorer created using C++.  
It allows the user to perform different file and directory operations directly from console using a menu driven interface.

It supports basic file operations along with some extra (novelty) features.

---

## Features

### Basic Features
- List files & folders
- Enter directory
- Go back (parent directory)
- Create file
- Delete file
- Copy file
- Move / Rename file
- Search files (recursive)
- Show file permissions
- Change file permissions
- Open and view file content

### Novelty Features (Added Extra)
- Write data into file
- Create directory
- Delete directory recursively
- Show file details (file size & last modified date)
